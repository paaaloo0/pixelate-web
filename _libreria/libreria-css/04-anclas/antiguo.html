Link Simple

Los vínculos en HTML tienen 4 estados que podemos manipular a través de CSS.

Estos estados son:

  • Normal → link
  • Visitado → visited
  • Sobre → hover
  • Activo → active

Para definir reglas para vínculos debemos utilizar un tipo especial de selector, llamado PSEUDO-SELECTOR.
