Hipervínculos

Los hipervínculos son el cimiento donde se sostiene la web

Introducción:

Los vínculos en la web se crean con la etiqueta <a> que viene de la palabra "anchor" que significa ancla.

El elemento "a" puede contener texto, una imagen y otros elementos HTML, pero nunca otros elementos "a". Sea cual sea el elemento, este se transformará en un elemento "cliqueable" y su estilo cambiará para indicar que es un link.

Las anclas equieren el atributo "href" y opcionalmente se pueden agregar otros tales como "title" o "target".

El atributo href le indica al navegador dónde debe dirigirse cuando el usuario haga clic en el elemento. Sin este atributo el ancla no se comportará como hipervínculo, sin embargo, seguirá siendo un ancla (ver vínculos del mismo documento para más información).

El atributo title crea una pequeño mensaje sobre el link al poner el cursor sobre, éste despliega el valor del atributo. "Target" en tanto, le indica al navegador si debe abrir el nuevo vínculo en la misma ventana o en una nueva

Existen 3 tipos de vínculos:

  • A otros sitios
  • Dentro del mismo sitio
  • Dentro del mismo documento

Recuerda que:

Los elementos ancla son elementos en linea.

Vínculos a otros sitios

Un link a otro sitio luce asi:

<a href="http://www.google.cl" title="Visitar Google.cl" target="_blank">Entrar a Google</a>

Este código resulta en:

Entrar a Google

Vínculos dentro del mismo sitio

Un link a un documento dentro del sitio sigue el mismo patrón que el anterior, la única diferencia es que el valor del atributo "href" es una ruta relativa y no absoluta:

<a href="pagina.html" title="Entra a una página de este sitio" target="_self">Entra a una página de este sitio</a>

Este código resulta en:

Entra a una página de este sitio

Vínculos dentro del mismo documento

Un link a un documento del mismo documento nos permite crear un vínculo que apunte a alguna sección dentro de la misma página donde nos encontramos.

En este proceso necesitamos dos elementos, el primero será el botón que apretemos para "saltar" a otra parte del documento. El segundo elemento es el destino del botoón marcado con un atributo "id" que nos guiará.

Seleccionando el destino

Para seleccionar el destino, agregamos un atributo "id" a éste de la siguiente manera:

<h1 id="inicio">Bienvenida</h1>

Si revisas este documento, veras que el título de la página tiene un "id" con valor inicio

Crear el vínculo

Para hacerlo, debes armar una etiqueta "ancla" normalmente, la diferencia es que el valor del atributo "href", debe ser el valor del "id" del elemento destino, antecedido del simbolo "#". Por ejemplo

<a href="#inicio">Pínchame para volver arriba</a>

Este código resulta en:

Pínchame para volver arriba

Vínculos con imágenes

Crear vínculos con imágenes sigue el mismo procedimiento que un link normal, la diferencia es que el contenido del link es una etiqueta <img>.

Ejemplo

<a href="http://www.facebook.com" title="Visítanos en Facebook" target="_blank"> <img src="facebook.png" alt="Logo Facebook" /> </a>
